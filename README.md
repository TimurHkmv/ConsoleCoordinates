# ConsoleCoordinates

Небольшая консольная утилита на C++: парсит точки из текста в формате `(n:n)` (или других скобок и других разделителей) и рисует их на координатной плоскости в ASCII, а также выводит расстояния от первой точки и сортирует их. 
## Сборка

```bash
mkdir build
cd build
cmake ..
cmake --build .
